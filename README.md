##  Project Overview
This project focuses on building an interactive Student Performance Dashboard using Microsoft Excel to analyze academic data and generate MIS-style reports.

##  Objective
To consolidate multi-sheet academic data into a structured master dataset and create a dynamic dashboard for performance monitoring.

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Data Validation
- SUMIFS, IF, AVERAGE formulas
- Conditional Formatting

##  Dataset Description
The dataset includes:
- Student Name
- Subject-wise Marks
- Study Time (Weekly)
- Attendance / Absence
- Strength & Weakness Indicator

##  Project Steps

1. Combined multiple subject sheets into a single Master Data table.
2. Cleaned and validated data (duplicates, formatting, structure).
3. Created Pivot Tables for subject-wise performance analysis.
4. Built interactive dashboard using:
   - Dropdown selection
   - Slicers
   - Dynamic charts
5. Designed visual reports:
   - Marks Distribution
   - Study Time Analysis
   - Absence Report
   - Strength vs Weakness Comparison

##  Key Insights
- Identified high-performing and low-performing subjects per student.
- Compared individual absence against overall average.
- Monitored study-time distribution.
- Generated dynamic MIS-style academic reports.

##  Outcome
Developed a fully interactive Excel dashboard suitable for MIS reporting and performance tracking.
